# Puertos Comunes y Métodos de Explotación para Máquinas Linux

| **Puerto** | **Servicio**              | **Descripción**                                                                  | **Herramientas/Métodos de Explotación**                                                                                                     |
|------------|---------------------------|----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| 22         | SSH                       | Protocolo para acceso remoto seguro.                                              | Fuerza bruta con `hydra` o `medusa`. Enumeración de versiones y configuraciones con `nmap` (`nmap -p 22 -sV --script ssh2-enum-algos`).                                  |
| 80         | HTTP                      | Servidor web.                                                                     | Escaneo de directorios con `gobuster` o `dirb`. Análisis de aplicaciones web con `nikto`, `Burp Suite` o `OWASP ZAP`.                                                  |
| 443        | HTTPS                     | Versión segura de HTTP usando TLS/SSL.                                           | Análisis de certificados SSL con `sslscan` o `testssl.sh`. Las aplicaciones web vulnerables pueden ser analizadas con herramientas como `Burp Suite` o `OWASP ZAP`.                   |
| 21         | FTP                       | Protocolo de Transferencia de Archivos.                                           | Acceso anónimo o credenciales débiles con `ftp`, `hydra` o `nmap` (`nmap -p 21 --script ftp-anon`).                                                              |
| 23         | Telnet                    | Protocolo de acceso remoto no cifrado.                                            | Enumeración y fuerza bruta con `hydra` o `medusa`. Captura de credenciales en texto plano con herramientas como `tcpdump` o `Wireshark`.                                    |
| 25         | SMTP                      | Protocolo para enviar correos electrónicos.                                       | Enumeración de usuarios con `smtp-user-enum`. Detección de reenvío abierto con `nmap` (`nmap -p 25 --script smtp-open-relay`).                                                  |
| 110        | POP3                      | Protocolo para recibir correos electrónicos.                                      | Escaneo de credenciales débiles con `hydra` o `medusa`.                                                                                                            |
| 143        | IMAP                      | Protocolo para acceder a correos electrónicos almacenados en un servidor.          | Enumeración y explotación de credenciales con `hydra`.                                                                                                              |
| 53         | DNS                       | Protocolo del Sistema de Nombres de Dominio.                                      | Enumeración de zonas DNS con `dig`, `nslookup` o `dnsenum`.                                                                                                         |
| 3306       | MySQL                     | Base de datos relacional.                                                        | Enumeración de credenciales con `mysql`, `hydra` o `nmap` (`nmap -p 3306 --script mysql-enum`).                                                                       |
| 5432       | PostgreSQL                | Base de datos relacional avanzada.                                               | Enumeración de credenciales con `psql`, `hydra` o `metasploit`.                                                                                                       |
| 6379       | Redis                     | Sistema de almacenamiento de datos en memoria.                                    | Acceso no autorizado con `redis-cli`. Escaneo con `nmap` (`nmap -p 6379 --script redis-info`).                                                                    |
| 2049       | NFS                       | Protocolo para compartir archivos a través de una red.                            | Enumeración de exportaciones con `showmount` o `nfs-common`.                                                                                                               |
| 8080      | Servidores HTTP Proxy/Alt | Servidores web alternativos o proxies.                                            | Escaneo con `nikto`, `gobuster` y análisis con `Burp Suite`.                                                                                                  |
| 3128       | Proxy Squid               | Proxy HTTP utilizado para filtrado de tráfico.                                    | Explotación de configuraciones débiles con `curl` o scripts personalizados.                                                                                                 |
| 5900       | VNC                       | Protocolo para compartir escritorio remoto.                                       | Enumeración y explotación de credenciales débiles con `nmap` (`nmap -p 5900 --script vnc-auth`). Fuerza bruta con `hydra`.                                         |
| 111        | RPCBind                   | Protocolo RPC utilizado para mapear servicios en un servidor.                     | Enumeración con `rpcinfo` y `nmap` (`nmap -p 111 --script rpcinfo`).                                                                                            |
| 69         | TFTP                      | Protocolo de Transferencia de Archivos Trivial.                                   | Enumeración y explotación de configuraciones con `tftp` o `nmap` (`nmap -p 69 --script tftp-enum`).                                                            |
| 8000/8080  | Servidores HTTP Alt/Proxy | Servidores web alternativos o APIs.                                               | Escaneo y enumeración con `gobuster`, `dirb`, y análisis de aplicaciones web con `Burp Suite`.                                                                  |
| 9000       | Gestión Web de Samba      | Interfaz web para gestionar los servicios de Samba.                               | Explotación de configuraciones vulnerables o credenciales débiles con `smbclient` y `metasploit`.                                                                    |
